How To Make Rocket Animation Using HTML & CSS


Letâ€™s start the article in this article we have to learn about how to make rocket animation using html & css.

Rocket Animation

  • Step 1 :

    Create a one text file & give to name this file like index.html

  • Step 2 :

    Create another new text file & give to name this file like style.css

  • step 3 :

    For this project we can required the rocket image so we can download it form the google.



  • Step 4 :

    Now open html file in the text editor which you can used & write the following code in it.

    • < !doctype html >

    < html lang="en" >

       < head >

          < meta charset="utf-8" >

          < title >Rocket Animation< /title >

          < link rel="stylesheet" type="text/css" href="style.css" media="screen" >

       < /head >

       < body >

          < div class="scene" >

             < div class="rocket" >

                < img src="rocket.jpg" >

             < /div >

          < /div >

          < script >

             function stars()

             {

                let count = 20;

                let scene = document.querySelector('.scene');

                let i = 0;

                while(i < count)

                {

                   let star = document.createElement('i');

                   let x = Math.floor(Math.random() * window.innerWidth);

                   let duration = Math.random() * 1;

                   let h = Math.random() * 100;

                   star.style.left = x + 'px';

                   star.style.width = 1 +'px';

                   star.style.height = 50 + h + 'px';

                   star.style.animationDuration = duration + 's';

                   scene.appendChild(star);

                   i++;

                }

             }

             stars();

          < /script >

       < /body >

    < /html >

  • step : 5

    And then open the css file & then write the following code.

    • *

    {

         margin : 0;

         padding : 0;

         box-sizing : border-box;

    }

    .scene

    {

         position: relative;

         width: 100%;

         height: 100vh;

         background: #000;

         overflow : hidden;

         display : flex;

         justify-content : center;

         align-items : center;

    }

    .scene i

    {

         position : absolute;

         top : -250px;

         background : rgba(255,255,255,0.5);

         animation : animateStars linear infinite;

    }

    @keyframes animateStars

    {

         0%

         {

                     transform: translateY(0);

         }

         100%

         {          transform: translateY(100vh);

         }

    }

    .scene .rocket

    {

         position : relative;

         animation : animate 0.2s ease infinite;

    }

    @keyframes animate

    {

         0%

         {

                     transform: translateY(-2px);

         }

         100%

         {          transform: translateY(-2px);

         }

    }

    .scene .rocket::before {

         content : " "

         position : absolute;

         bottom : -200px;

         left : 50%;

         transform : translateX(-50%);

         width : 10px;

         height : 200px;

         background : linear-gradient(#00d0ff,transparent);

    }

    .scene .rocket::after {

         content : " "

         position : absolute;

         bottom : -200px;

         left : 50%;

         transform : translateX(-50%);

         width : 10px;

         height : 200px;

         background : linear-gradient(#00d0ff,transparent);

         filter : blur(20px);

    }

  • step : 6
    And save this file & the open this html file in chrome & see the output like that.
